# Bank Teller Crisis

## Overview
A Python banking simulation project that demonstrates account management operations such as balance inquiry, deposits, withdrawals, and account-to-account transfers.

## Features
- Check account balances
- Deposit funds
- Withdraw funds
- Transfer funds between accounts
- Input validation
- Error handling

## Technologies
- Python 3

## Concepts Used
- Functions
- Conditional Statements
- Variables
- String Formatting
- Business Logic

## Final Results
Initial Balance:
Checking = $0
Savings = $0

Final Balance:
Checking = $35
Savings = $5
